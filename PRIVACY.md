# OrderRx Training Track — 개인정보 처리방침 (Privacy Policy)

- **문서 버전**: 1.0.0
- **최종 수정일**: 2026-07-13
- **적용 대상**: Chrome 확장프로그램 "OrderRx Training Track"

본 확장은 약국 도매 주문 사이트에서의 업무 흐름(workflow)을 연구하기 위한
**참여 동의 기반 데이터 수집 도구**입니다. 이 문서는 확장이 무엇을 수집하고,
무엇을 수집하지 않으며, 수집된 데이터가 어떻게 처리되는지를 설명합니다.
확장 첫 실행 시 표시되는 참여 동의서(`apps/extension/CONSENT.md`)와 내용이 일치합니다.

## 1. 수집하는 데이터

참여자가 **직접 등록하고 브라우저 권한을 허가한 사이트**에서, **수집 활성화 상태일 때만**
아래의 행동 메타데이터를 수집합니다.

- 페이지 방문·이동 기록 (세션 토큰, 개인정보 패턴이 제거된 정제 URL)
- 클릭한 요소의 구조 정보 (CSS selector, 태그, ARIA role, 표시 텍스트 앞 40자 — 민감 패턴 치환 후)
- 입력 필드 사용 메타데이터 (필드 이름·종류, 입력 **길이**, 문자 유형 분류 — **값 원문 절대 미수집**)
- 폼 제출 메타데이터 (필드 개수, 필드 이름 목록 — 민감 필드명은 마스킹)
- 화면에 결과가 표시된 시점과 항목 개수
- 참여자 구분용 무작위 ID(UUID), 세션 ID, 확장 버전, 화면 크기

## 2. 수집하지 않는 데이터 (자동 차단)

다음은 저장 전 단계에서 자동 차단·마스킹되며, 어떤 형태로도 수집·전송되지 않습니다.

- 비밀번호, OTP (비밀번호 필드는 길이조차 기록하지 않음)
- 입력 값의 원문 (검색어 포함)
- 결제 정보 (카드번호 등), 주민등록번호
- 환자 개인정보, 전화번호, 이메일 (화면 텍스트에 나타나면 `[PHONE]` 등으로 치환)
- 쿠키, 세션 토큰, 인증 정보
- 실명, 브라우저 계정 정보, 등록하지 않은 사이트에서의 어떤 활동도 수집하지 않음

## 3. 데이터의 저장과 전송

1. 수집된 이벤트는 먼저 참여자 브라우저(IndexedDB)에 저장됩니다.
2. 주기적으로 운영자가 관리하는 수집 서버(Supabase, 대한민국 서울 리전)에 HTTPS로 업로드됩니다.
3. 업로드가 확인된 데이터는 브라우저에서 자동 삭제됩니다.
4. 서버 접근 권한은 쓰기 전용으로 제한되어, 참여자 단말에서 다른 참여자의 데이터를 읽을 수 없습니다.

## 4. 이용 목적과 제3자 제공

- 데이터는 **약국 주문 workflow 분석과 자동화 소프트웨어(모델) 개발**에만 사용됩니다.
- 데이터를 판매하지 않으며, 광고·신용평가 등 무관한 목적으로 사용하지 않습니다.
- 원본 데이터를 제3자에게 제공하거나 공개 저장소에 게시하지 않습니다.

## 5. 보관과 삭제

- 보관 기간: 프로젝트 종료 또는 참여자의 철회 요청 시까지.
- 참여자는 언제든 popup에서 수집을 중단하거나 로컬 데이터를 삭제할 수 있습니다.
- 참여 철회 시 participant_id를 운영자에게 알려주면 서버의 해당 데이터 전부를 7일 내 삭제합니다.

## 6. 동의

확장은 첫 실행 시 동의 화면을 표시하며, 명시적 동의 전에는 어떤 데이터도 수집·전송하지 않습니다.
동의 시각·동의서 버전·확장 버전이 기록됩니다.
