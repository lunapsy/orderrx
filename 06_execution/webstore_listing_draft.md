## Single Purpose (심사 핵심 — 한 문장)

> 참여 동의한 약국 테스터가 직접 등록한 주문 사이트에서, 업무 흐름(workflow) 연구를 위한
> 행동 메타데이터(클릭·페이지 이동·폼 사용 기록)를 수집하는 연구 참여 도구입니다.

## 상세 설명 (스토어 표시용)

```
OrderRx Training Track은 약국 주문 업무 자동화 연구에 참여하는 테스터 전용 데이터 수집 도구입니다.
일반 사용자를 위한 확장이 아니며, 운영자에게 안내받은 참여자만 사용합니다.

■ 어떻게 동작하나요
- 첫 실행 시 참여 동의서가 표시되며, 동의 전에는 어떤 데이터도 수집하지 않습니다.
- 참여자가 직접 등록하고 브라우저 권한을 허가한 사이트에서만 동작합니다.
- 수집되는 것은 행동 메타데이터뿐입니다: 어떤 페이지에서 무엇을 클릭했고,
  어떤 입력창을 사용했는지(입력 길이만), 폼을 언제 제출했는지.

■ 절대 수집하지 않는 것 (자동 차단)
- 비밀번호, OTP, 입력한 값의 원문
- 카드번호 등 결제 정보, 주민등록번호, 환자 개인정보
- 전화번호·이메일(자동 마스킹), 쿠키·세션 토큰
- 등록하지 않은 사이트에서의 모든 활동

■ 참여자 통제권
- 언제든 일시 정지 / 사이트별 중단 / 전체 삭제 가능
- 수집된 데이터를 popup에서 직접 확인·내보내기 가능
- 참여 철회 시 서버 데이터 전부 삭제(7일 내)

소스 코드 공개: https://github.com/lunapsy/orderrx
```

## 권한 정당화 (Privacy practices 탭)

| 권한 | 정당화 문구 |
|---|---|
| `storage` | 참여자 설정(동의 기록, 등록 도메인, 수집 ON/OFF)과 업로드 대기 이벤트를 저장합니다. |
| `scripting` | 참여자가 등록·허가한 도메인에만 수집 스크립트를 동적으로 주입합니다. 등록하지 않은 사이트에는 주입되지 않습니다. |
| `alarms` | 수집된 이벤트를 1분 주기로 수집 서버에 업로드하기 위한 타이머입니다. |
| `activeTab` | popup에서 현재 탭의 도메인을 확인해 등록을 돕는 용도입니다. |
| optional host permissions (`https://*/*`, `http://*/*`) | 참여자가 어떤 약국 주문 사이트를 쓰는지 사전에 알 수 없으므로, **설치 시에는 어떤 사이트 권한도 요청하지 않고** 참여자가 도메인을 등록하는 순간 해당 도메인에 한해 런타임 권한을 요청합니다. |
| 원격 코드 | 사용하지 않음 (모든 코드는 패키지에 포함). |

## 데이터 사용 공시 (Data usage 폼 체크 항목)

- 수집: **사용자 활동** (클릭·페이지 탐색 메타데이터), **웹 기록** (등록 사이트의 정제된 URL)
- 수집 안 함: 개인 식별 정보(실명·주소·이메일·전화), 건강 정보, 금융·결제 정보, 인증 정보, 개인 통신, 위치, 키 입력 원문
- 인증서: ✅ 판매하지 않음 / ✅ single purpose와 무관한 용도로 사용·양도하지 않음 / ✅ 신용도 판단·대출 목적으로 사용하지 않음
- 데이터는 HTTPS로 전송, 운영자 서버(Supabase 서울)에 저장, 명시적 사전 동의 후에만 수집
