# Privacy and Data Collection Policy Notes

## 원칙
- 최소 수집
- 목적 제한
- 로컬 우선
- 업로드 전 마스킹
- 철회 가능
- 삭제 요청 가능

## 절대 수집 금지
- 아이디/비밀번호/OTP 원문
- 카드/계좌/결제정보
- 환자명/연락처/주소 등 개인정보
- 쿠키/세션토큰

## 허용 가능 범위
- 사이트 ID
- 페이지 유형
- 이벤트 종류
- selector 의미 정보
- 성공/실패 상태
- 입력 길이/분류/해시

## 보관 원칙
- 파일럿 목적 달성에 필요한 최소 기간만 보관
- 삭제 요청 시 상태 추적 가능해야 함
