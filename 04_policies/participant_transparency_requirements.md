# Participant Transparency Requirements

Training Track 사용자는 아래를 반드시 할 수 있어야 한다.

## 필수 기능
- 현재 수집 중인지 확인
- 어떤 사이트에서 수집이 활성화되는지 확인
- 최근 수집 이벤트 미리보기 확인
- 수집 항목/비수집 항목 설명 보기
- 전체 Pause / Resume
- 사이트별 ON / OFF
- 참여 철회
- 삭제 요청 제출 및 상태 확인

## UX 원칙
- 중단 버튼은 숨기지 않는다.
- 수집 재개는 명시적 사용자 행동으로만 가능하다.
- 미리보기에는 민감정보 원문이 보이면 안 된다.
