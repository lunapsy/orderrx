# Glossary

- **Training Track**: 파일럿 참여자를 위한 확장프로그램 기반 수집 트랙
- **Execution Track**: 실제 자동화 실행 트랙
- **Site Adapter**: 사이트별 DOM 구조와 workflow를 다루는 규격
- **Deterministic Executor**: LLM 없이도 정의된 규칙대로 동작하는 실행기
- **LLM Abstraction Layer**: 모델 공급자 차이를 숨기는 추상 계층
- **Transparency Dashboard**: 사용자가 수집되는 이벤트를 확인하는 화면
- **Site-off**: 특정 사이트에 대한 수집 비활성화
- **Withdraw**: 참여 철회
- **Delete Request**: 업로드된 데이터 삭제 요청
- **Fallback Distribution**: Web Store 공개 승인 실패 시 대체 배포 방식
